# Vocabuddy

Vocabuddy 是一个支持中文、英文、西里尔蒙古文互学的个性化网页端背单词工具。

## 当前项目文档

- [产品需求文档（PRD）](docs/PRD.md)

PRD 已明确：

- 用户注册后的主要母语设置
- 母语优先的界面、解释和学习反馈
- 目标语言与学习目的配置
- 学习水平评估、每日计划和达成预测
- 新词、间隔复习、生词本和测验闭环
- 跨母语好友互相推荐单词
- 好友推荐任务、确认机制和推荐理由翻译
- 好友挑战、进度对比和隐私设置

## 状态

当前仓库处于产品定义阶段，下一步进入技术方案和 MVP 开发。

详见 [docs/PRD.md](docs/PRD.md)。
